This is a program to find "paths" between words.
You can only change 1 letter in one step, to get another correct word.
Something like "папа" -> "лапа" -> "лама" -> "мама"
To run this program, just run
java Main <from word> <to word>
It will print the shortest way from first to second word or information that there is no such way.
Program works with Russian dictionary.
Have fun!
